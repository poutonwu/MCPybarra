## 问题与建议

### 主要问题：

1. **部分异常处理不够优雅**：
   - 如 `search-collections` 的 `SearchWithZeroLimit` 返回 400 错误，建议自动调整为默认 limit 值（如 10）。
   
2. **集合接口返回 404 问题**：
   - `get-collection-info` 的有效 ID 请求返回 404，可能表示集合 API 接口存在问题或尚未完全实现。

3. **部分响应内容截断**：
   - 虽然不是功能缺陷，但建议增加日志记录或前端提示以避免误解。

### 改进建议：

- 对边界参数进行默认值兜底处理；
- 增加对集合 API 的完整支持；
- 增强文档说明，尤其是关于参数限制和行为定义；
- 增加更多安全测试覆盖（如 CSRF、CSRF Token 等）。
